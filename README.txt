Приложение

   - в качестве входящей информации получает:
	url страницы официального сайта Государственная Дума/Состав и структура Гос. Думы/Состав Гос. Думы седьмого созыва
	http://www.duma.gov.ru/structure/deputies/

   - считает количество и средний возраст депутатов

Языки: Python3

Модули, библиотеки Python: urllib, re, BeautifulSoup4

Среда разработки: Jupyter Notebook, Anaconda, Командная строка, Notepad++ на Windows

Приложение состоит из:
    - gosduma.py

Презентация 'Python.pptx' содержит снимки экранов результатов работы приложения в командной строке, 
фрагментов страниц/html, используемых приложением, а также DOM этих фрагментов.

На момент тестирования приложения на официальном сайте Государственной Думы была информация о 447 депутатах.

Примерное время работы приложения для подсчёта количества депутатов и их среднего возраста - 4.5мин.
